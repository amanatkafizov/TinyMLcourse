# TinyMLcourse
This TinyML course focuses on deploying machine learning models on resource-constrained IoT edge devices, rather than teaching machine learning theory itself. The emphasis is on practical implementation—how to take pre-trained models and optimize, compress, and run them efficiently on microcontrollers and embedded systems.

Participants will learn how to bridge the gap between AI and hardware by working with real-world tools and workflows for model quantization, memory optimization, and low-power inference. The course highlights challenges unique to edge environments, such as limited compute, energy constraints, and real-time operation.

By the end, learners will understand how to integrate intelligent functionality directly into IoT devices, enabling applications like smart sensing, anomaly detection, and on-device decision-making—without relying on cloud-based processing.
